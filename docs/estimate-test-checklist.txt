File:    src/ai/flows/generate-painting-estimate.exterior.ts:328-336

재현:

  roofArea = 50 × 1.3 = 65 sqm

  roofMax = 65 × 42 = 2730

  최종 범위 폭이 0, min == max.

원인:
  Roof에 고정 floor($3,500)가 있는데, rate × area가 floor보다 작을 때
  max도 floor로 올려서 min == max가 됨.

영향:
  사용자에게 단일 가격을 "Range"로 표시 → 모델의 리스크 신호 소실.
  소형 지붕 + 단층 조합에서 자주 발생 예상.

제안:
  roofMax에도 별도 ceiling buffer (예: floor × 1.25) 적용해 최소 폭 확보.
    const effectiveRoofFloorMax = roofRate.floor * 1.25;
    roofMax = Math.max(roofArea * roofRate.max, effectiveRoofFloorMax);
